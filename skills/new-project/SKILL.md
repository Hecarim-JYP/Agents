# 새 프로젝트 세팅

**`~/.claude/jyp/scaffolds/default.md`가 절차의 유일한 출처(SSOT)다** — 파일 전체를 읽고 그 절차(시작 결정 체크리스트 확인 → 구조 생성 → git 초기화 → 보고)를 그대로 수행한다.

- 구조·초기 파일·언어별 조정의 세부를 이 문서에 요약하지 않는다 — 요약이 원본과 어긋나는 드리프트를 방지 (스캐폴드 문서만 갱신하면 된다).
- 핵심 주의만 재확인: 이미 파일이 있는 폴더에는 생성하지 않고 사용자에게 먼저 확인, 요청받지 않은 샘플 코드 생성 금지.
